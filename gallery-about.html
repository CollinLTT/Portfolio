About


This gallery website was made as a test of my CSS skill progression.

As I worked through my Web Development course I came upon an optional project. The project was a simple photo gallery containing a site title, a bar under the header, and square images spaced evenly. I decided this would be a great opportunity to see how my CSS skills have improved, to implement responsive CSS, and to take the prompt a step further.

I ended up creating a gallery with the authors of the photos displayed as images are hovered, a better styling of the images, custom text selection colors, a navigation bar, and custom animations.
